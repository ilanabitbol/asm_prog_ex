Basic fasm Answers
==================

Notes:
The answers are stored as seperate files, starting with the corresponding 
question number.

0.  Write a program the receives three numbers and sums those three numbers.
    Then output the result.


1.  Write a program that receives two numbers a,b and calculates their integral
    average.


2.  Write a program that receives a number x as input, and outputs to the
    console the following values: x+1, x^2 , x^3, one after another.


3.  Write a program that receives two numbers a,b as input, and outputs the
    remainder of dividing a by b.
